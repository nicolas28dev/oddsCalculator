User Guide

1. Enter the number of total tickets and the ticket price.

2. For each prize amount, add a prize by clicking the "Add Prize" button and enter the prize amount and the number of winning tickets for that prize.

3. Click the "Calculate Odds" button to see the expected value and expected winnings based on the information you provided.

The calculator computes the expected value and winnings by multiplying the probability of winning each prize by the prize amount, then summing up these values. The expected value represents the average amount you would win if you played the game an infinite number of times, while the expected winnings show your potential winnings for the specified number of tickets you entered.
